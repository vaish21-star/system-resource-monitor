# System Resource Monitor (Python)

## Description
A Python-based system monitoring tool that displays real-time system information such as CPU usage, memory usage, disk usage, and top running processes.

## Features
- Displays operating system details
- Shows CPU usage with alert system
- Displays memory statistics
- Shows disk usage
- Lists top 5 running processes by memory usage

## Technologies Used
- Python
- psutil
- platform module

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python monitor.py
